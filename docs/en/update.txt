To update the Wordpress version follow these steps:

++ Git

Make sure you're on master

<geshi type="bash">
git status
# if needed
git checkout master
</geshi>

++ Update Wordpress

<geshi type="bash">
cd /path/to/modules/Wordpress/htdocs
rm -rf content
svn co http://core.svn.wordpress.org/tags/3.1.3 content
</geshi>

++ Update version

Set the version in the Module

<geshi type="bash">
vi lib/Wordpress/Module.php
</geshi>

<geshi type="php">
/**
 * current wordpress version
 */
const WORDPRESS_VERSION = '3.1.3';
</geshi>


++ Git it

<geshi type="bash">
git status
# check changes
...
git commit -a -m "Updated wordpress to 3.1.3"
git tag wordpress-3.1.3
git push --tags
</geshi>
